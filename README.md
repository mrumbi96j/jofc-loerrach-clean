# Juventus Official Fan Club Lörrach G. Agnelli

Pacchetto pronto da caricare su GitHub e pubblicare su Vercel.

## Come usarlo

1. Carica tutti i file di questa cartella su un repository GitHub.
2. Vai su Vercel e importa il repository.
3. Vercel riconoscerà automaticamente Vite.
4. Premi Deploy.

## Comandi locali

```bash
npm install
npm run dev
```

## Nota

Le partite attuali sono demo. Il modulo biglietti apre una e-mail precompilata verso `jcdgagnelli@gmx.de`.
